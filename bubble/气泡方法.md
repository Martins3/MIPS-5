# 插入气泡具体表示
ID 段的指令如果与 EX、 MEM 或 WB 段的指令存在数据冲突，则 ID 段
正在执行的指令暂停一个时钟周期（通过阻塞 IF/ID， PC 实现）， 时钟到来时将 ID/EX 流水接
口部件同步清零，也就是在 EX 段插入一条空指令（气泡）， 如果此时 ID 段仍然 MEM、 WB 段数
据相关，则继续阻塞 IF/ID 接口部件， 下一个时钟到来时继续在 EX 段插入一个气泡， 依次类
推直至....
